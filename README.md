# 🌐 多智能体舆情干预衰减仿真平台

基于SEIR传播模型和指数衰减理论的舆情干预效果仿真系统

## ✨ 核心功能

- **多智能体仿真**: 基于Mesa框架的Agent建模，支持KOL、活跃用户、普通用户三种类型
- **SEIR传播模型**: 模拟舆情从易感→潜伏→传播→恢复的完整生命周期
- **干预衰减建模**: 指数衰减公式 `E(t) = Σ δₖ·e^(-λ(t-tₖ))`，支持多次干预的信任折损
- **社交网络**: 支持小世界、无标度、随机三种网络拓扑
- **可视化分析**: 基于Plotly的交互式图表，实时展示仿真结果

## 🎯 功能模块

- 运行单次SEIR仿真

- 可视化社交网络状态

## 🔧 参数说明

### 人群参数
- **Agent总数**: 仿真中的智能体数量
- **KOL比例**: 意见领袖占总数的比例
- **初始感染数**: 仿真开始时的传播者数量

### 传播参数
- **传播率 β**: 控制舆情传播速度
- **潜伏→传播率 σ**: 潜伏者转为传播者的概率
- **自然恢复率 γ**: 无干预情况下的自然恢复率
- **网络拓扑**: small_world(小世界) / scale_free(无标度) / random(随机)

### 干预参数
- **初始干预强度 δ₀**: 首次干预的基础效果 (0~1)
- **衰减系数 λ**: 效果衰减速度，越大衰减越快
- **信任折损 α**: 多次干预时的强度折损系数 (0~1)
- **干预触发时刻**: 在第几步触发干预

## 📊 衰减公式

单次干预效果：
```
E(t) = δ₀ · e^(-λ·t)
```

多次干预叠加：
```
E(t) = Σ δₖ · e^(-λ·(t-tₖ))
其中 δₖ = δ₀ · α^(k-1)
```

## 💡 使用提示

1. **参数调整**: 建议从默认参数开始，逐步调整观察效果
2. **网络规模**: Agent数量过大(>500)会影响性能，尤其是网络快照
3. **干预时机**: 通常在传播高峰前干预效果最佳
4. **多次干预**: 观察信任折损对后续干预效果的影响
